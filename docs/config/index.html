General Site & Build Settings

Site Settings

These settings control the basic site details:

View example's code 
# Site Settings
baseurl: ""
url: ""

Build Settings

Configure Jekyll build options:

View example's code 
# Build settings
markdown: kramdown
permalink: /blog/:title/
sass:
  style: compressed
  sass_dir: _sass
  sourcemap: never

Plugins

These plugins extend Jekyll functionality:

View example's code 
# Plugins
plugins:
  - jekyll-feed
  - jekyll-seo-tag
  - jekyll-sitemap
  - jekyll-paginate-v2
  - jekyll-watch
  - jekyll-toc

TOC Settings

Configure Table of Contents settings:

View example's code 
# TOC Settings for documentation
toc:
  min_level: 2
  max_level: 3
  ordered: false
  numbered_items: true
  auto_ids: true
  sanitize: false

Content & Functionality Settings

Pagination

Settings for blog and project pagination:

View example's code 
# Pagination
pagination:
  enabled: true
  collections:
    - name: 'posts'
      per_page: 6
      permalink: '/blog/page:num/'

Collections

Define collections for projects and authors:

View example's code 
# Collections
collections:
  projects:
    output: true
    permalink: /projects/:path/
  authors:
    output: true
    permalink: /authors/:path/

LiveReload

Settings for automatic page reloading:

View example's code 
# LiveReload settings
livereload_max_delay: 5
livereload_min_delay: 2.5
